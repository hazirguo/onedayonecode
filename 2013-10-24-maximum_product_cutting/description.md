# 高效切绳子 #

## 题目描述 ##

有一个长度为 n 米的绳子, 将绳子切成不同段的整数长, 有多种切割的方法, 将切成的整数相乘, 求最大的乘积. 假设绳子长大于 2 米, 你至少得切一刀.

举例:

* 输入 n = 2, 输出 1 (1*1)
* 输入 n = 3, 输出 2 (1*2)
* 输入 n = 4, 输出 4 (2*2)
* 输入 n = 10, 输出 36 (3*3*4)
